NOTES

We have 4 main components in table

These elements don't help or hinder the accessibility of the table, but they are useful in terms of usability. They provide styling hooks.

  • Caption (OPTIONAL)
  • tHead same as th (OPTIONAL)
  • tbody, which represents the body of the table (OPTIONAL)
  • tfoot, which represents the foot of table (OPTIONAL)

Tables can be divided into table headers, bodies, and footers, but none of these really does anything if the tables do not contain table rows, cells, and content. Each table row, <tr> contains one or more cells. If a cell is a header cell, use <th>. Otherwise, use <td>
